# Social Bookmarking Platform

A social platform built with Django where users can share images and interact with each other. Users can bookmark any image on the internet and share it with others. They can also see activity from the users they follow and like/unlike the images shared by them.

## Features

- **User Authentication**
  - Login and Logout views using Django's authentication framework.
  - Password change and reset functionality.

- **User Registration**
  - Custom user registration views to allow new users to sign up.

- **User Profile**
  - Extended user model with a custom profile model to store additional user information.
  - Configured the project to handle media file uploads.

- **Image Sharing**
  - Users can share images and bookmark any image on the internet.
  - Users can like or unlike images shared by others.

- **Activity Feed**
  - Users can see activity from the users they follow, including images liked or shared by them.
